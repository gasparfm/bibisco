How to install bibisco
1. Drag bibisco to the Application folder
2. Hold down the control key while clicking in this window and select Eject "bibisco" from the menu.
3. For El Capitan and previous versions, on System Preferences -> Security and Privacy, you must have "Allow apps downloaded from" set to "Anywhere.". 

Note: For Sierra you have to launch bibisco with Secondary Click (Right Click or ‚åÉCTRL+Click) in Finder and choosing open.

Note: bibisco is a java application and ships its own JRE. You don't need to install or update JRE.
